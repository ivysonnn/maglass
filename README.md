# maglass

## About

**maglass** is a screen magnifier based on Tsoding's **[Boomer](https://github.com/tsoding/boomer)**, but reimplemented in C using **[Raylib](https://github.com/raysan5/raylib)**.

## Dependencies

- **C11+**
- **CMake 3.16+**
- **Grim** (for taking screenshots on Wayland)
- **[Raylib](https://github.com/raysan5/raylib)**

## Installation

### Cloning

```bash
git clone https://github.com/ivysonnn/maglass
cd maglass
```

### Installing

There's a Shell Script for doing so.

```bash
./install.sh
```

If the installation script doesn't work, you'll have to install the dependencies manually.
